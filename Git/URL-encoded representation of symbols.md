# URL-encoded representation of Symbols

1. Space: `%20`
2. Exclamation mark ( ! ): `%21`
3. Double quotation mark ( " ): `%22`
4. Number sign ( # ): `%23`
5. Dollar sign ( $ ): `%24`
6. Percent sign ( % ): `%25`
7. Ampersand ( & ): `%26`
8. Single quotation mark ( ' ): `%27`
9. Left parenthesis ( ( ): `%28`
10. Right parenthesis ( ) ): `%29`
11. Asterisk ( * ): `%2A`
12. Plus sign ( + ): `%2B`
13. Comma ( , ): `%2C`
14. Forward slash ( / ): `%2F`
15. Colon ( : ): `%3A`
16. Semicolon ( ; ): `%3B`
17. Less than ( < ): `%3C`
18. Equal sign ( = ): `%3D`
19. Greater than ( > ): `%3E`
20. Question mark ( ? ): `%3F`
21. At symbol ( @ ): `%40`
